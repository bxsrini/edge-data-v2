# ONELIGN Edge Data V2

IoT Sensor Monitoring Platform — Landing Dashboard Redesign

## Overview
This repository contains the redesigned Landing Dashboard for the ONELIGN sensor monitoring platform, built with React, TypeScript, Tailwind CSS, and Recharts.

## Getting Started
```bash
npm install
npm run dev
```

## Tech Stack
- React 18 + TypeScript
- Vite
- Tailwind CSS
- Recharts
- Lucide React Icons
